Installation of geopandas on windows machines:
https://towardsdatascience.com/geopandas-installation-the-easy-way-for-windows-31a666b3610f

1. Go to https://www.lfd.uci.edu/~gohlke/pythonlibs/ (Unofficial Windows Binaries for Python Extension Packages)

2. Download on a specific folder the following binaries (if not already installed):
* GDAL
* Pyproj
* Fiona
* Shapely
* Geopandas
* Pystan
* Prophet

The binaries must match the version of Python and whether the 32-bit or 64-bit OS is installed.

(E.g., for Python v3.9x (64-bit), GDAL package should be GDAL-3.4.1-cp39-cp39-win_amd64.whl)

3. Install the files in the following order:
* GDAL
* Pyproj
* Fiona
* Shapely
* Geopandas
* Pystan
* Prophet

(E.g., pip install <dir>\GDAL-3.4.1-cp39-cp39-win_amd64.whl)
